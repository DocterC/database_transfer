# fastadmin适配达梦数据库

使用方法：
  1.将thinkphp/connector/Dm.php保存到项目中，路径：\think\db\connector目录下。
  2.将thinkphp/builder/Dm.php保存到项目中，路径：\think\db\builder目录下。
  3.修改env中数据库配置：
  [database]
  hostname = 127.0.0.1
  username = VN
  database = VN
  password = ******
  hostport = 5236
  prefix = fa_
  type = dm
